Markdown 语法

前言

建站的第一件事就是先去学习了一下 Markdown 语法 , 方便对于博客的编写 。

Markdown 对于文本的编辑与 HTML 相比可以说是非常的通俗易懂 , 下面是我对与 Markdown 常用语法使用的一些总结 。

标题

# ~ ###### 一到六级标题

# 一级标题

以此类推

换行

在一行的末尾添加两个或多个空格 ，然后按回车键 , 即可创建一个换行 。

强调

斜体： *斜体* ， 效果：斜体

粗体： **粗体** ， 效果：粗体

粗体+斜体：***粗体+斜体 *** ， 效果：粗体+斜体

删除线 ~~删除线~~ ，效果： 删除线

引用

在段落前添加一个 > 符号 ，可嵌套其他元素 。

> this is my blog

this is my blog

  • welcome!

列表语法

  1. 有序列表 ，可嵌套

    1. 标题
      1. 一级标题
    2. 换行
      1. br

  1. 无序列表

    - title
    - br

    • title
    • title

代码

`markdown` ，markdown

``using namespace `std` `` ，using namespace `std`

代码块：将代码块的每一行缩进至少四个空格或一个制表符。

typedef struct node{
    int x = x;
}

``` C++

代码围栏区域

```

链接

超链接：

[超链接显示名](超链接地址 “超链接title”)

Jia

网站链接：

< * https://markdown.com.cn * >

< https://markdown.com.cn >

可内嵌格式化链接，粗体，斜体，代码

图片链接：

![图片alt](图片链接 “图片title”)

转义字符

格式化 Markdown 文档的字符，请在字符前面添加反斜杠字符 \ 。

表格

（—-）创建每列的标题，并使用管道（|）分隔每列。

| Syntax | Description |
| —————- | —————- |
| Header | Title |
| Paragraph | Text |
